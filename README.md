## Getting Started

First download by yarn

```
yarn

```

Run the development server:

```
yarn dev

```

## Setup env file

I provide a example .env.example file

```
NEXT_PUBLIC_API_URL=
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=

BUNNY_CDN_STORAGE_URL=
BUNNY_CDN_USERNAME=
BUNNY_CDN_ACCESS_KEY=
BUNNY_CDN_URL=
```
